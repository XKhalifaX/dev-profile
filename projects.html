Project 02

Math Dungeon

A browser-based math combat game where players solve timed equations to defeat monsters, manage health, and survive escalating difficulty.

HTML CSS JavaScript Game Logic

Equation combat preview, timer pressure, and health-bar feedback in action.

Core Architecture

  • Centralized game state for player health, monster health, current answer, and difficulty
  • Vanilla JS gameplay loop: generate equation → submit answer → apply damage → update UI
  • Difficulty-based monster generation and progression toward a final boss
  • Real-time UI updates for announcements, equation text, monster image, timer bar, and health bars

The Technical Challenge

Balancing game feel across speed, difficulty, and feedback required careful tuning of timers, damage scaling, and progression logic. I solved this by structuring gameplay into reusable functions, updating state and UI in one loop, and using time-based damage/reward rules so faster correct answers feel meaningfully stronger without breaking difficulty pacing.
